# SARTI
Sistema Administrador de Recursos de TI.
Version Framework 2.0 (Debido a que el servidor donde se monto la solución cuenta con desarrollos en esa version)
-----
                                                Almacenamiento de archivos.
Carpeta /Facturas_Equipos: Almacena las facturas de los equipos, cargadas al momento de dar de alta un equipo.
  ** Formato en el que se almacena el archivo: "FACTURA_" <numFactura> "NSerie_" <numSerie> <tipoExtención (.pdf - .jpg - .png)>
Carpeta /Responsivas_Equipos: Almacena la responsiva firmada por el usuario al que se le asigno un equipo.
  ** Formato en el que se almacena el archivo: "RESPONSIVA_nSerie_" <numSerie> "user_" <numUser> "Equipo_" <idEquipo> <extension (.pdf - .jpg - .png)>
